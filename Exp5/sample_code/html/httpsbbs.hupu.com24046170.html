虎扑10月23日讯 2013年10月20日，孙悦婚礼隆重举行，“大圣”与娇美新娘情定终生。眨眼间时光如白驹过隙，近日孙悦爱妻陈露在社交媒体上晒出二人浓情嘟嘴甜美合影，纪念二人的“木婚”时光。

孙悦和陈露一直被称作是篮球界的恩爱代表，两人从相恋到结婚，再到如今组建四口之家，感情始终很好，秀恩爱也成为了他们平日里的常态。在孙悦的职业生涯中，遭遇了不少质疑，作为孙悦爱妻的陈露，一直力挺老公，是孙悦的坚强后盾。

孙悦连续两个赛季无缘北京队的大名单后，他决定赴美特训一年，明年将以更加出色的状态来迎接新赛季的到来。他最近一次出现在大众视野，是与爱妻陈露在上海出席了NBA冠军主题展。
