公司简介

茵曼（INMAN）是广州市汇美时尚集团股份有限公司旗下全资品牌，由公司董事长方建华于2008年创立。茵曼以“素雅而简洁、个性而不张扬”的品牌形象风格诞生于互联网，更专注于产品品质与消费者体验的打造。产品均选用最优质、环保、舒适的棉和麻作为面料，设计取材来自江南水乡、国画、水墨、印花等中国文化精髓，让消费群体切身体验与世无争脱离都市喧哗，倡导自然“慢生活”的品牌主张。
